## Description courte (132 chars max — actuel : 87)
Connecte le chat Zoom, Meet, Teams et WebinarJam aux overlays MDI Live en temps réel.

## Description longue

MDI Live Watchtower est l'extension Chrome officielle de la plateforme MDI Live.

Elle capte les messages du chat de vos sessions en ligne et les transmet instantanément aux overlays interactifs affichés dans OBS Studio — sans aucune manipulation pendant votre animation.

**Ce que fait l'extension :**
- Détecte les votes Quiz (A / B / C / D) et les envoie à l'overlay Sondage
- Alimente le Nuage de mots en temps réel
- Comptabilise les votes pour le Tug of War, le Décompte Bonhomme, le Décompte Poker et le Mot Magique
- Transmet les commentaires longs à l'overlay Commentaires
- Fonctionne en arrière-plan, sans interrompre votre présentation

**Plateformes supportées :**
- Zoom (zoom.us)
- Google Meet (meet.google.com)
- Microsoft Teams (teams.microsoft.com / teams.live.com)
- WebinarJam et EverWebinar

**Usage réservé :**
Cette extension est réservée aux clients de la plateforme MDI Live. Un identifiant de salle (Room ID) est nécessaire pour l'activer.
