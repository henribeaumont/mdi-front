## Long description

MDI Live Watchtower is the official Chrome extension for the MDI Live platform.

It captures chat messages from your online sessions and instantly forwards them to interactive overlays displayed in OBS Studio — with no manual action required during your presentation.

**What the extension does:**
- Detects Quiz votes (A / B / C / D) and sends them to the Poll overlay
- Feeds the Word Cloud in real time
- Counts votes for Tug of War, Countdown (Stick Figure), Poker Countdown, and Magic Word overlays
- Forwards long comments to the Comments overlay
- Runs silently in the background without interrupting your presentation

**Supported platforms:**
- Zoom (zoom.us)
- Google Meet (meet.google.com)
- Microsoft Teams (teams.microsoft.com / teams.live.com)
- WebinarJam and EverWebinar

**Restricted use:**
This extension is reserved for MDI Live platform clients. A Room ID is required to activate it.
